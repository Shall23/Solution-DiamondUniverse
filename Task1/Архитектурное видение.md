Проблематика 
Архитектура поможет в определенные сроки (в рамках ТЗ это 2 месяца), запустить промежуточное решение, которое решает БТ в части запуска цифрового аукциона без существенных доработок уже имеющихся систем в компании «Diamond Universe». По "рельсам" промежуточного решения, будет разработано целевое решение 
Решение данной задачи 

Контекст

С помощью новой архитектурной концепции, компания планирует в кратчайщие сроки реализовать первый аукцион, для продажи алмазов в онлайн режиме
Ключевые участники БП: 

Представители компании
Эксперты
Организациторы аукциона
РП
В рамках MVP планируется заключение контракта с провайдером частного облака, для реализации публикации данных для клиентов

Описание оптимального решения

Для MVP необходимы организационные решения в части публикации данных для клиентов в формате excel в облачное хранилище по определенным шаблонам. Сроки реализации данного решения укладываются в ожидания заказчика (2 месяца) 
Для целевого решения, необходимо сформировать концептуальную архитектуру проекта, презентовать ее всем стейкехолдерам. На основании данных действий, можно определить сроки реализации проекта в зависимости от свободных ресурсов (загруженность текущих команд в части аналитиков, разработчиков, тестировщиков, РП и др)
