Список систем
ИС «Алмазы» (существующая система): отбор алмазов для цифрового аукциона (новый), планирование работ, присвоение идентификаторов и подготовка ведомостей. - Необходимо реализовать алгоритм по отбору алмазов 
ИС «Логистика» (существующая система): отправка алмазов в лабораторию, и возврат в офис после оцифровки (новый процесс).Перевозка алмазов, в т. ч. в лабораторию (новый процесс) и покупателям. - Необходимо разработать новый сервис по отправки алмазов в лабораторию и возврат их в офис после оцифровки 
ИС «Аукцион» (существующая система) : публикация данных для клиентов (новый процесс), проведение аукциона. - Необходимо на этапе промежуточного решения реализовать публикацию данных в облако с данными в для клиентов в формате в EXCEL. Для целевого решения необходима передача данных в новую систему "ЛК клиента".
ИС «Бухгалтерия» (существующая система): актирование и оплата услуг лаборатории (новый процесс), компаний-перевозчиков, ведение контрактов. - Доработка текущей системы и создание нового сервиса для оплаты услуг лаборатории и актирование 
ИС «Лаборатория» (новая система): оцифровка алмазов
ИС «Личный кабинет клиента» (новая система): Отображение данных для клиентов. Отслеживание проведение аукциона - Создание новой системы с оределенными функциями системы 


Интеграции                
ИС «Алмазы» → ИС «Логистика»: планирование и перевозка алмазов. Существующая интеграция.
ИС «Лаборатория» → ИС «Аукцион»: передача описания алмазов (новая). Автоматическая. Новые данные 
ИС «Лаборатория» → ИС «Бухгалтерия»: актирование и оплата услуг лаборатории (новая). Автоматическая. Новые данные 
ИС «Логистика» → ИС «Бухгалтерия»: актирование и оплата услуг транспортной компании.
ИС «Аукцион» → облачный сервис. Данные для клиентов. Новая интеграция для промежуточного решения. Ручной режим
ИС «Аукцион» → ИС «Личный кабинет клиента». Новая интеграция для целевого решения. Автоматический режим. Новые данные
